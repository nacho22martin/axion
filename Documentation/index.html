General

Flattern is a flat & trendy HTML site template built with responsive Twitter bootstrap. Please remember, it's HTML CSS site template so you need to edit with any HTML editor to customize text or content. This template coming with "Boxed" and "Wide (default)" layout and has 15 pre-made template accent colors as you can see on folder 'color'. We've separated each accent color to a single css file to make you easy to edit template color without touching other css files.

You can go to http://twitter.github.com/bootstrap/ if you want to learn more about twitter boostrap since this is the main css framework of Flattern template.

Again, since it's HTML CSS based you can edit all text, content with HTML editor like notepad++ or anykind else. Quite easy for people with basic HTML-CSS knowledge :)

Use background pattern images or customize body background (applied on boxed mode)

When you use boxed wrapper, you'll be able to show your body backgound. We've included 10 repeatable background images in folder 'img/bodybg' that can be used as body background.

Currently each body background called by single css file, see in folder 'bodybg' there are 10 css files from bg1.css.... to bg10.css

see css code example (from bg1.css) which call "bg1.jpg" as body background

body {
    background-image: url(../img/bodybg/bg1.jpg);
    background-repeat: repeat;
}
 
say you want to use bg1.jpg pattern image as body background, so just call the bg1.css from your HTML file and put before </head> tag
 
    <!-- boxed bg -->
    <link id="bodybg" href="bodybg/bg1.css" rel="stylesheet" type="text/css" />
 
If you don't want to use this way and has HTML-CSS knowledge you can also edit from 'style.css' directly, find "body" tag and put background value
 
body {
    font-family:'Open Sans', Arial, sans-serif;
    font-size:13px;
    line-height:1.6em;
    color:#656565;
   <!-- your body background -->
    background-image: url(../img/bodybg/bg1.jpg);
    background-repeat: repeat;
}
 

Use premade template color

Flattern has 10 premade accent colors that can be found in folder 'color', by default each HTML file uses 'default.css' (orange color).

if you want to use other color, just open your html file and  find following code (on line 18-19)

 

    <!-- Theme skin -->
    <link href="color/default.css" rel="stylesheet" />
 
just change 'default.css' above with other available color e.g 'blue.css', 'yellow.css'..so on from folder 'color'
 
 

Template font / typography

This template using 2 main colors from Google webfont

1. Open sans (Body and heading)

2. Noto serif (for italic style used on blockquote, testimonial)

Both font above imported via "style.css" on line 1-2

 

/* ==== Google font ==== */
@import url('http://fonts.googleapis.com/css?family=Noto+Serif:400,400italic,700|Open+Sans:300,400,600,700');

Element settings

Nivo slider

used on index.html

all slider images for index.html stored in folder 'img/slides/nivo' .You can overwrite example images with your own images, for easiest way just use same name, image format and size so you don't need to edit the HTML

You can set nivo slider parameter from the Nivo slider function that written in 'custom.js' file (see folder js) as follows :

 

        //nivo slider
        $('.nivo-slider').nivoSlider({
            effect: 'random', // Specify sets like: 'fold,fade,sliceDown'
            slices: 15, // For slice animations
            boxCols: 8, // For box animations
            boxRows: 4, // For box animations
            animSpeed: 500, // Slide transition speed
            pauseTime: 5000, // How long each slide will show
            startSlide: 0, // Set starting Slide (0 index)
            directionNav: true, // Next & Prev navigation
            controlNav: false, // 1,2,3... navigation
            controlNavThumbs: false, // Use thumbnails for Control Nav
            pauseOnHover: true, // Stop animation while hovering
            manualAdvance: false, // Force manual transitions
            prevText: '', // Prev directionNav text
            nextText: '', // Next directionNav text
            randomStart: false, // Start on a random slide
            beforeChange: function(){}, // Triggers before a slide transition
            afterChange: function(){}, // Triggers after a slide transition
            slideshowEnd: function(){}, // Triggers after all slides have been shown
            lastSlide: function(){}, // Triggers when last slide is shown
            afterLoad: function(){} // Triggers when slider has loaded
        });

Portfolio

3 columns --> Portfolio li class: <li class="item-thumbs span4 design" data-id="id-0" data-type="web">

Contact form

As described in our description, we use working PHP contact form on 'contact.html'. All PHP files for the contact form located in 'contact' folder. To make the contact form work properly you'll be required to edit config.php and place your email address in the place of the default value: "name@email.com". That's all needed to run the contact form.
